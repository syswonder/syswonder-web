# 矽望开源社区

## 矽望社区

矽望泛在操作系统开源社区（Syswonder Community） 是一个非营利性、商业中立的开源技术社区，旨在研究适用于未来泛在智能应用场景，包括具身智能、智能制造等现场计算场景的新型泛在操作系统关键技术。社区以合力打造具有国际影响力的先进开源项目为目标，倡导不同单位的研究人员、工程技术人员围绕共建开源项目开展开放、协作的技术研究与创新。

**社区关注**

社区面向需要在近场进行实时感知和态势认知的现场计算场景打造的项目包括：

* 面向边缘异构计算平台，Rust实现、经形式化验证的轻量高安全虚拟机监控器[hvisor](https://github.com/syswonder/hvisor)
* Rust实现的一体内核Unikernel操作系统[ruxos](https://github.com/syswonder/ruxos)
* 具身智能操作系统 [robonix](https://github.com/syswonder/robonix)

**社区治理**

社区采用开放、透明、民主协商的治理方式：

* 技术委员会（TC）: 负责技术路线、项目管理和技术决策。
* 开源社区委员会: 负责社区生态建设、应用推广和社区运营。
* 社区事务管理员: 负责日常事务处理和基础设施运维。


## 矽望开源技术委员会

矽望开源技术委员会定义和决策社区技术愿景和路线，建设和管理社区核心项目。
