# Syswonder Open Source Community

The **Syswonder Ubiquitous Operating System Open Source Community** (the **Syswonder Community**) is a **non-profit, vendor-neutral open source technical community** dedicated to the research and development of key technologies for next-generation ubiquitous operating systems.

The Community focuses on future ubiquitous intelligent computing scenarios, including **embodied artificial intelligence**, **intelligent manufacturing**, and other **in-situ computing environments**. Its goal is to collaboratively build advanced open source projects with international impact, and to foster **open, cooperative research and innovation** among researchers and engineering professionals from diverse organizations through the joint development of open source projects.

**Focus Areas**

The Community develops open source projects targeting **in-situ computing scenarios** that require real-time perception and situational awareness in near-field environments, including but not limited to:

- **hvisor**: a lightweight hypervisor for heterogeneous edge computing platforms, implemented in Rust and formally verified  
  <https://github.com/syswonder/hvisor>

- **ruxos**: an Unikernel operating system implemented in Rust  
  <https://github.com/syswonder/ruxos>

- **robonix**: an operating system for embodied intelligence  
  <https://github.com/syswonder/robonix>

**Community Governance**

The Community adopts an **open, transparent, and consensus-driven governance model**, consisting of the following bodies and roles:

- **Technical Committee (TC)**: Responsible for defining technical directions, managing projects, and making technical decisions.
- **Open Source Community Committee**: Responsible for ecosystem development, application outreach, and community operations.
- **Community Administrators**: Responsible for day-to-day operations and infrastructure maintenance.


## Technical Committee

The Syswonder Technical Committee defines and maintains the
community's technical vision, guides and builds core community projects.
